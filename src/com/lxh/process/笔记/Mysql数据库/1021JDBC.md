         ```java
                 }
         ```

      4. 释放资源

         ```java
             public static void closeConnection(AutoCloseable autoCloseable){
                 try {
                     if (autoCloseable != null){
                         autoCloseable.close();
                     }
                 }catch (Exception e){
                     e.printStackTrace();
                 }
             }
         ```
